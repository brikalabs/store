# Boite a outils i18n

Traduisez, formatez et localisez vos contenus dans plus de 40 langues,
directement depuis votre hub Brika. La boite a outils embarque un catalogue de
messages hors ligne pour l'anglais, le francais, l'allemand et l'espagnol, et
peut appeler un service de traduction automatique si vous fournissez une cle
d'API.

## Outils

- **translate** — traduire une phrase d'une langue vers une autre.
- **detect-language** — deviner la langue d'un texte.

## Blocs

- **Localize message** — un bloc de transformation qui resout une cle de message
  pour la locale active, avec repli sur votre langue par defaut.

## Confidentialite

Sans cle d'API, toutes les traductions sont effectuees localement et aucun texte
ne quitte votre hub.
