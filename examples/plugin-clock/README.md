# World Clock

Timezone-aware clocks, timers, and scheduled sparks for your Brika hub.

## Tools

- **now** - get the current time in any IANA timezone.

## Blocks

- **Every interval** - a trigger block that emits a tick on a fixed interval.

## Sparks

- **tick** - fired every configured interval.
- **alarm** - fired when a scheduled alarm is due.

## Pages

- **clocks** - a custom tab showing all your configured clocks at a glance.
